### 📋 Pré-requisitos

- [Node.js](https://nodejs.org/)

### 🔧 Instalação

#### Clone project
```
git clone git@github.com:LucasKinoshita/premium-store.git
```

#### Install
```
yarn install
```

#### Start application
```
yarn dev
```

#### Start api
```
yarn start:api
```

## ⚙️ Executando os testes

```
yarn test
```
