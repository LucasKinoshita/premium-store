### 🚀 Features
- Add to cart
- Cart order
- Product listing

### 📋 Requirements

- [Node.js](https://nodejs.org/)

### 🔧 Instalação

#### Clone project
```
git clone git@github.com:LucasKinoshita/premium-store.git
```

#### Install
```
yarn install
```

#### Start application
```
yarn dev
```

#### Start api
```
yarn start:api
```

## ⚙️ Running the tests

```
yarn test
```
